## User Journey Visualization

### Sign Up with Google (New User)

```
User visits page
      โ
Sees "Sign up with Google" button (white, prominent at top)
      โ
Clicks button โ Button shows "Connecting..."
      โ
Redirected to Google login page
      โ
User enters Google credentials (or already logged in)
      โ
Google asks "Allow Social Media Automator to..."
      โ
User clicks "Allow"
      โ
Redirected back to your app's dashboard
      โ
Workspace created automatically
      โ
See dashboard โ Ready to start posting!

โฑ๏ธ  TOTAL TIME: 30 seconds (or 5 seconds if already logged into Google)
```

```

      โ

      โ

      โ

      โ
      โ INSTANT redirect to dashboard
      โ
      โฑ๏ธ  TOTAL TIME: 5 seconds

      โ
      โ Redirected to Google login
      โ
      โ Enter Google credentials
      โ
      โ Redirected to dashboard
      โ
      โฑ๏ธ  TOTAL TIME: 20 seconds
```
